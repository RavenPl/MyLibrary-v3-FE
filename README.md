### :heavy_exclamation_mark: Prerequisites

This project uses npm as package manager

```bash  
 npm install --global npm  
```  
<br/>
<!-- Run Locally -->  

# :running: Run Locally

Clone the project

```bash  
 git clone https://github.com/RavenPl/MyLibrary-v3-FE.git
```  

Go to the project directory

```bash  
 cd my-library-FE  
```  

Install dependencies

```bash  
 npm install  
```  

Start the server

```bash  
 npm start  
```
